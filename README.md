# 기능요구사항

- 로또 구입금액을 입력하면 구입금액에 해당하는 로또를 발급해야 한다.
- 로또 1장의가격은 1000원이다.
- 로또 당첨 금액은 고정되어 있는 것으로 가정한다.
- 수익률을 계산해 출력해야 한다.


## 구현 해야 할 기능목록
- 랜덤한 6자리의 숫자를 발행한다. 보너스번호 1개 
- 로또 번호는 1~45 숫자 이다.
- 구입한 로또번호를, 저장해야 한다.
- 저장해 놓은 로또번호를, WinningLotto와 비교한다.
- 비교한 값을 저장한다.
   - 저장시 Rank에 잇는 등수별로 맞춘 갯수를 저장
- 로또 수익률을 계산한다.
